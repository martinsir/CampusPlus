# CampusPlus
Campus+ App skal modernisere uddannelsesmessen ved at samle tilmelding, materialer og evaluering ét sted. Fokus er på lærere, der kan registrere klasser, fordele elever i grupper og administrere billetter, mens studerende får adgang til aktivitetspas og undervisningsmaterialer. Appen skal være klar til CAMPUS+ i 2025.
